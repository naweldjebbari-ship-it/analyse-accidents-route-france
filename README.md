# analyse-accidents-route-france
Analyse exploratoire et dashboard Power BI sur les accidents de la route en France

## Objectif
Analyser les facteurs influençant la gravité des accidents de la route en France.

## Outils utilisés
- Python
- pandas
- Power BI
- DAX
- SQL

## Étapes du projet
- Nettoyage et structuration des données
- Analyse exploratoire
- Création de KPIs
- Visualisation avec Power BI

## Résultats
- Analyse temporelle des accidents
- Analyse géographique
- Étude du taux de mortalité
- Identification de zones à risque
